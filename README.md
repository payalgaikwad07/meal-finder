# Meal Finder 🍽️

A minimal working website built with Node.js, Express, and EJS that integrates TheMealDB API to search and discover meals.

## How to Run

1. Clone the repo
   git clone https://github.com/payalgaikwad07/meal-finder.git

2. Install dependencies
   npm i

3. Start the server
   node index.js

4. Open in browser
   http://localhost:3000

## Features
- Search meals by name
- Browse meals by cuisine
- Get a random meal
- View full recipe with ingredients and YouTube link

## API Used
TheMealDB - https://www.themealdb.com/api.php

## Tech Stack
- Node.js
- Express.js
- Axios
- EJS
